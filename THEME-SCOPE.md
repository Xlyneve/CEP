# CEP theme scope

The shared Ashen Rose theme is intentionally excluded from these local pages:

- `biosched1.html` — Biologics Planner
- `Notes.html` — standalone Notes
- `recalls.html` — Recalls
- `recalltracker.html` — Recall Tracker
- `recall-notes.html` — Recall Notes

These external applications are also outside the CEP theme scope and must not be changed as part of CEP visual work:

- OH Cold Chain
- Expiry Check
- PDRP

Do not add `site-background.css` to the protected local pages unless the user explicitly changes this decision.
